## üìÑ Description 
This is the API for a social media web application where users can share their thoughts, react to friends‚Äô thoughts, and create a friend list. It uses Express.js for routing, a MongoDB database, and the Mongoose ODM. In addition to using the Express.js and Mongoose packages. Users can interact with API routes through Insomnia.

To verify the code on your local machine clone the code and run 

`npm install`
